Interactive region-based segmentation demo: click a pixel to grow a region from it based on intensity similarity

  • Each cell is a "pixel" with a grayscale intensity (a synthetic image with two brighter blobs on a noisy dark background).
  • Click any cell to pick it as a seed.
  • The algorithm grows outward to neighboring pixels (4-connectivity), adding any neighbor whose intensity is within the threshold of the seed's value — the core idea of region-based segmentation: start from a seed and merge neighboring pixels that are similar enough.
  • Raise or lower the threshold to see the region grow larger (more lenient) or stay tight (stricter).
